# Project Lichen

Observing.
Learning.
Sharing.

Project Lichen is an evolving collection of practical observations from living landscapes.

Project Lichen is also a framework for reconnecting living systems, energy systems, human coordination, land use, technology and governance into one coherent design. We already have many pieces in place and working.

It explores regenerative horticulture, seed stewardship, biodiversity, fungi, soils, pollinators, recipes, climate resilience and ecological participation.

The project values evidence over certainty and curiosity over conclusions.
